# POWER-BI-HR-REPORT-BY-SQL-QUERY-

# 🏢 HR Report Data — SQL + Power BI Dashboard

An interactive HR analytics dashboard built with Microsoft Power BI,
powered by SQL-cleaned data to uncover workforce insights.

## 📊 Dashboard Overview

The dashboard provides a comprehensive view of key HR metrics including:

- **Count of Job Title by Gender & Month** — Area chart tracking hiring trends
- **Count of Job Title by Emp ID** — Donut chart showing employee distribution
- **Count of Department by First Name** — Pie chart for department breakdown
- **Count of Location City by Job Title** — Bar chart for city-wise job roles
- **Count of Race & Job Title by City & Year** — Clustered bar chart for diversity trends
- **City Wise Filter** — Slicer for regional data exploration

## 🗄️ Data Workflow

Raw HR Data → **SQL Cleaning & Transformation** → Export CSV → **Power BI Dashboard**

## 🗂️ Data Fields Used

- Employee: emp_id, first_name, gender, race, birthdate
- Job Info: job_title, department, hire_date
- Location: location_city, location_state

## 🛠️ Tools & Technologies

- Microsoft Power BI Desktop
- MySQL — Data Cleaning & Querying
- DAX (Data Analysis Expressions)
- Custom Dark Purple Theme
